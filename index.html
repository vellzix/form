Rangkuman HTML Basic Tag

Pengenalan HTML

HTML adalah bahasa markup untuk membuat halaman web dan menampilkan informasi di browser.

Sejarah HTML

  • HTML 2.0 (1995)
  • HTML 3.2 (1996)
  • HTML 4.0 (1997)
  • XHTML 1.0 (2000)
  • XHTML 1.1 (2007)

Ciri-Ciri Dokumen HTML

  • Ekstensi file .htm atau .html
  • Non-case sensitive
  • Terdiri dari tag pembuka dan penutup
  • Tag saling berpasangan dan bersarang

Struktur HTML

Struktur dasar dokumen HTML terdiri dari:

  • <html>: Elemen utama dokumen
  • <head>: Informasi tentang dokumen
  • <body>: Konten yang ditampilkan

Tag Dasar HTML

Tag penting dalam HTML mencakup:

  • Heading: <h1> sampai <h6>
  • Paragraf: <p>
  • Garis horizontal: <hr> dan baris baru: <br>
  • Gambar: <img>
  • Hyperlink: <a>
